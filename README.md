# 安裝方式

全域安裝
`npm i webpack@4.43.0 -g`
`npm i webpack-cli@3.3.12 -g`

1. 要裝在 c 槽
2. 安裝4.43.0 版本  npm i webpack@4.43.0 -g
3. 查詢版本 webpack -v
4. node 升級到 12.19.0 版本( current目前的版本 )


```bash
webpack-cli 4.1.0

webpack 4.43.0

```
# 產生 package.json

 `npm init`  <br>


# 安裝在專案裡
`npm install webpack@4.43.0 --save-dev` <br>
`npm install webpack-cli@3.3.12 --save-dev` <br>
